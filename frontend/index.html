ER-модель

Концептуальный уровень — сущности, идентификаторы, связи. ER-модель не оперирует ключами.

Модель семантических объектов

Концептуальный уровень — именованные наборы атрибутов. Кардинальность: min.max. ID — идентификатор.

Реляционная модель

Логический уровень — отношения, первичные ключи, внешние ключи, нормализация (3НФ / БКНФ).

PK — первичный ключ
FK — внешний ключ
UK — ключ-кандидат (AK)
GEN — вычисляемый столбец
── FK → PK (стрелка ссылки)

Стрелки соответствуют связям FK → PK.
